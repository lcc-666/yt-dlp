# yt-dlp
一个基于yt-dlp的python程序，在yt-dlp上进行了简单的封装，将不常用的功能封装，只留下一些常用的功能。

# 环境准备
建议使用[python官网](https://www.python.org/)3.7版本+[ffmpeg](https://ffmpeg.org/)

# 客户端环境
博主再win和Linux上进行了基本测试mac未经过测试。

#  文件安装
本程序基于yt-dlp
pip install yt-dlp

git clone https://github.com/lcc-666/yt-dlp.git
